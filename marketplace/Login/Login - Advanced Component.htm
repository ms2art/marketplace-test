Login - Advanced Component

Overview

Login - Advanced is a component that contains a login screen with text fields for the user name and password, and a submit button. The component supports social authentication from identity providers such as Facebook, Office365, and LinkedIn. The component also supports Fingerprint Authentication for Android.
